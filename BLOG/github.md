# 创建一个仓库
# 与本地连接
# 本地上传

创建一个仓库在GitHub上进行操作

与本地连接：将仓库的连接复制

git remote add origin "http:/"

常用命令：

git status: 查看状态

git add . ：提交全部文件到暂存区

git add 具体文件名 ：提交某一文件

git commit -m“增加了新内容”：提交上去

git log : 查看所有的版本信息

git show 12738120310（就是将某一版本的那一串复制过来，shift+inset粘贴）查看某一具体的版本信息


本地上传：进入该文件夹

git add .

git commit -m "msg"

git push
